* HOW TO USE: To use this Plain-ish Jane To-do List, input and update to-do list items using the provided text fields. The Change Todo and Toggle Completed options have a smaller text field. These smaller text fields are used to select an item in the to-do list by selecting the number in which the item appears on the list. The first list item is considered item 0, so keep that in mind when choosing items to update on your list.
